# Blog MongoDB – Aggregation Pipeline Lab

Projeto didático para aprendizado de Aggregation Pipeline com MongoDB usando Flask.

## Objetivos
- Conexão com MongoDB Atlas
- Seed de dados
- Aggregation Pipeline real
- Consulta em tempo real via painel admin

## Execução
1. Configure sua string MongoDB em config.py
2. Execute o seed:
   python seed/seed_db.py
3. Inicie a aplicação:
   python app.py
